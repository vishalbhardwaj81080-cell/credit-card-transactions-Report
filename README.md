# credit-card-transactions-Report
This project presents an interactive Power BI dashboard built using a real-world dataset to analyze customer behavior and key business metrics. It includes data cleaning, transformation, and insightful visualizations such as KPIs, trend analysis, and category-wise performance to support data-driven decision-making.


 Project Overview

This project focuses on building an interactive and insightful Power BI dashboard using a structured dataset to analyze customer and business performance. The dashboard is designed to help stakeholders quickly understand key trends, patterns, and risk indicators through effective visual storytelling.



 Objectives
 
	•	Analyze customer-level and transactional data
	•	Identify key performance indicators (KPIs)
	•	Discover trends, patterns, and risk factors
	•	Create an interactive and user-friendly dashboard



  Data Cleaning & Preparation

 The following data preparation steps were performed:
	•	Handling missing and null values
	•	Correcting data types (date, numeric, categorical)
	•	Removing duplicates and invalid records
	•	Creating calculated columns and measures using DAX



 Dashboard Features

The Power BI dashboard includes:
	•	KPI Cards (Total Customers, Revenue, Risk Metrics, etc.)
	•	Category-wise and segment-wise analysis
	•	Trend analysis using line and bar charts
	•	Distribution analysis with pie/donut charts
	•	Interactive slicers for dynamic filtering



 Tools & Technologies 
 
	•	Power BI – Dashboard creation & visualization
	•	Power Query – Data transformation
	•	DAX – Measures and calculated fields
	•	CSV Dataset – Source data



 Key Insights
 
	•	Identified high-risk and low-performing segments
	•	Observed trends across different customer categories
	•	Enabled interactive exploration for better decision-making

Conclusion

This project demonstrates practical skills in data cleaning, analysis, and visualization using Power BI. It is suitable for data analyst / business analyst portfolio projects and showcases the ability to convert raw data into meaningful insights.
